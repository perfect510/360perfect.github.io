ABOUT US

The site provides relevant product information in one easy-to-use platform along with a mix of engaging content. Users can shop for products, read independent reviews from actual buyers. Our website does not sell products but instead, links consumers directly with manufacturer’s most current product data then connects these users directly with the manufacturer’s authorized retailers. Our website provides the users with a LIST OF SHOPS that provides with all the modification parts for customizing their vehicle. Also an option to BUY and SELL their vehicles(*ONLY MODIFIED VEHICLES). The user will also get to use the feature of “LIVE CUSTOMIZATION” where the user can customize their car based on the type that the user have selected. Based on the customization done by the user, our website will display, all the shops that provides with the modification parts that the user needs.
